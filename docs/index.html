per-app-isolation

Per-app isolation

Each web app gets its own cookie jar, local storage, and dedicated WebView profile on Android 13+. No cross-site tracking between your apps — by design, not by accident.

Android 13+ WebView profiles
argon2id + aes-256-gcm

Encrypted everywhere

Database encrypted with SQLCipher (AES-256). Per-app cookie jars encrypted between sessions (AES-256-GCM). Backup files protected by Argon2id + AES-256-GCM. Cryptographic keys backed by Android Keystore (hardware-backed where available).

Hardware-backed keys
built-in ad blocker

Built-in ad blocking

Per-app content blocker with custom rule support. Clean browsing experience without needing a separate app or browser extension. Toggle it on or off per app at any time.

Custom rules per app
system webview · geckoview

Two browser engines

Use Android System WebView for speed or optionally download Mozilla GeckoView for an independent engine. Switchable per app without reinstalling. User-agent can also be set per app.

Switch engines per app
